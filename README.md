🚀 **Full-Stack E-Commerce Application (FastAPI + React)**

This repository contains a modern, scalable e-commerce platform built using **FastAPI** for the backend and **React** for the frontend. It leverages **PostgreSQL** as the database, integrated via **SQLAlchemy ORM** for efficient data handling and modeling.

### 🔑 Key Features

* 🔐 **Authentication & Authorization**

  * Secure user authentication using **JWT (JSON Web Tokens)**
  * Role-based access control (RBAC)
* 🛒 **E-Commerce Functionality**

  * Product listing, search, and filtering
  * Cart and order management
* ⚡ **FastAPI Backend**

  * High-performance async APIs
  * Clean and modular architecture
* 🎨 **React Frontend**

  * Responsive UI with modern design patterns
  * API integration for seamless user experience
* 🗄️ **Database Layer**

  * PostgreSQL with SQLAlchemy ORM
  * Structured models and migrations support

### 🧱 Tech Stack

* **Backend:** FastAPI, SQLAlchemy
* **Frontend:** React.js
* **Database:** PostgreSQL
* **Auth:** JWT-based authentication

### 📦 Goal

To provide a production-ready foundation for building scalable e-commerce platforms with modern web technologies.
